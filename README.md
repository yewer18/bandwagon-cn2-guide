## 先搞懂：CN2 到底是什么

CN2 是中国电信运营的国际骨干网络，英文全称是 China Next Generation Carrier Network，分两个主要等级：

**CN2 GT（Global Transit，AS4809）**：比普通 163 骨干网（AS4134）贵，但 2019 年之后拥堵问题越来越明显，晚高峰高峰期丢包不罕见，属于"贵但没那么好"的一档。

**CN2 GIA（Global Internet Access，AS4809 高等级）**：这才是真正的企业级线路。搬瓦工官方的描述是"观察到问题最少的网络，非常稳定"。用来开视频会议、连 SSH、跑建站流量都没问题。代价是带宽极度稀缺，搬瓦工自己在洛杉矶部署了 8 条 10Gbps 的 CN2 GIA/CTGNet 链路来保证供应。

还有个 **CTGNet（AS23764）**，是电信的新补充线路，搬瓦工说它的表现和价格实际上和 CN2 GIA 等价。

讲真，如果目标用户主要在大陆，普通 163 线路晚高峰丢包率可以到 30% 以上——这种程度连正常浏览网页都费劲，更别说其他应用了。CN2 GIA 就是为了解决这个问题而存在的。
